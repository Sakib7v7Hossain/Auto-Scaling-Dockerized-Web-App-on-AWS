# Production-Ready Scalable AWS Architecture with Auto Scaling, ALB & WAF 🚀

## 📌 Overview
Designed and deployed a production-style scalable cloud infrastructure on AWS with automatic scaling, load balancing, and security integration.

---

## ⚙️ Architecture Highlights
- Automated EC2 provisioning using Launch Templates with Dockerized application deployment via user data.
- Configured Auto Scaling Group (min: 1, max: 3) with CPU-based scaling and integrated Application Load Balancer (ALB) for high availability.
- Implemented AWS WAF and validated system performance through stress testing, achieving dynamic scaling and zero downtime.

---

## 🛠️ Tech Stack
- **AWS Services:** EC2, Auto Scaling Group (ASG), Application Load Balancer (ALB), WAF  
- **Tools & Technologies:** Docker, Nginx, GitHub, Bash (user data scripts)  

---

## 🔥 Key Features
- Self-healing infrastructure with automatic scaling
- Secure traffic handling using AWS WAF
- High availability with load-balanced architecture
- Real-world simulation using stress testing

---

## 📈 Outcome
- Successfully demonstrated dynamic scaling under load  
- Achieved zero-downtime deployment  
- Validated production-like cloud architecture behavior
